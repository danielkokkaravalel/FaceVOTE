FaceVOTE

Login and start voting

A simple face id voting app created using Flask and Bulma, where students can register themselves. Once registered, student can login and cast their vote.


Candidate info tab gives you information about the candidates standing for the elections.


A new candidate can be registered only by the Admin. Admin can log in and fill a form to register a new candidate for the election.


The system uses face recognition for authenticating users. The system uses PostgresSQL Database deployed on AWS.
